# 天机 AI · WPS 智能办公助手\n\n<p align="center">\n  <strong>你的文档，自有天机</strong>\n</p>\n\n---\n\n## 简介\n\n**天机 AI**（TianJi AI）是一款专为 WPS Office 打造的智能办公插件，深度融合 AI 能力，让文档处理更智能、更高效。\n\n> "天机"取自《易经》——洞察文档玄机，化繁为简。\n\n## ✨ 功能特性\n\n### 🤖 AI 助手\n- 智能对话，随时解答办公疑问\n- 多模型支持：智谱 GLM-5/4.7/4.6、OpenClaw 等\n- Agent 模式：自动调用工具完成复杂任务\n\n### 📝 文档审校\n- **双模式切换**：修订标记模式 / 直接替换模式\n- 自定义审阅人名称，修订标记个性化显示\n- 支持自定义审校要求，一键记住常用设置\n\n### ⚖️ 智能审查\n- **广告法合规审查**：智能识别违规用语\n- **合同审查**：条款风险识别、缺失项提示\n- **预包装食品标签审查**：法规合规检测\n- **审查意见书**：正式法律文书风格输出\n\n### 🎨 用户体验\n- Ribbon 集成，原生办公体验\n- Windows & macOS 双平台支持\n\n## 📦 快速安装\n\n### 最新版本：v0.2.2\n\n1. 下载 [](https://github.com/iam-zekai/tianji-ai-release/releases/download/v0.2.2/TianJi-AI-v0.2.2.zip)\n2. 解压后运行安装脚本（Windows:  / macOS: ）\n3. 打开 WPS，在 Ribbon 栏找到「天机 AI」选项卡\n\n> 安装脚本会自动清理旧版本，无需手动卸载。\n\n## 🔄 更新日志\n\n### v0.2.2 (2026-03-02)\n- 代码重构优化，降低维护复杂度\n- 修复白屏问题：3 层防护机制\n- 修复 Chrome 90 兼容性\n- 路由懒加载失败自动重试\n\n### v0.2.0 (2026-02-28) - 品牌升级\n- 🎨 全新品牌：KAI 办公助手 → 天机 AI\n- 📄 审查意见书正式版上线\n\n[查看完整更新日志 →](../../releases)\n\n## 🛠 技术架构\n\n- **前端**: Vue 3 + Vite\n- **UI**: Refined Theme\n- **WPS 集成**: WPS Office JS API\n- **AI 引擎**: 多模型适配层（智谱 GLM / OpenClaw）\n\n## 📋 兼容性\n\n| 平台 | 最低版本 | 状态 |\n|------|----------|------|\n| WPS Windows | 2019+ | ✅ 完全支持 |\n| WPS macOS | 3.0+ | ✅ 完全支持 |\n| Chrome 内核 | 90+ | ✅ 兼容 |\n\n## 📝 许可证\n\nMIT License © 2026 天机 AI Team\n\n---\n\n<p align="center">\n  遇到问题？请提交 <a href="https://github.com/iam-zekai/tianji-ai-release/issues">Issue</a>\n</p>
